# Politique de confidentialité - SnapFlash

Dernière mise à jour : 22 mai 2026

## Collecte des données
SnapFlash ne collecte aucune donnée personnelle.

## Caméra
L'application utilise la caméra uniquement pour prendre des photos avec des filtres. Aucune image n'est stockée sur nos serveurs.

## Partage des données
Nous ne vendons ni ne partageons vos données avec des tiers.
